# 3조 서버 프로그램 구현 ( BOOK SERVICE )

# 조원 👥
- 이재용
- 배주현
- 김준영

# STACKS 😶‍🌫️
- 프론트엔드 : HTML, CSS, JS, JSP
- 백엔드 : SPRING BOOT
- 데이터베이스 : MYSQL, JPA, MYBATIS

# 주요 기능
- 도서 대여, 대여 목록 조회, 반납, 반납일 연장 기능
- 도서 회원 가입(회원등록), 회원 정보 조회, 회원 정보 수정, 회원 정보 삭제
